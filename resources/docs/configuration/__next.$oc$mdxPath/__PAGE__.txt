# 配置参考

达梦 MCP 服务支持通过命令行参数、环境变量和 `.env` 文件配置。配置优先级从高到低：

1. 命令行参数
2. 环境变量
3. `.env` 文件（支持 `.env.local`、`.env.prod`）
4. Docker secrets
5. 默认值

## 配置格式

- **命令行**：`--` 前缀，嵌套用 `.` 分隔，如 `--server.port 18081`
- **环境变量**：大写 + `__` 分隔嵌套，如 `SERVER__PORT=18081`
- **`.env` 文件**：格式与环境变量相同

## 应用密钥（必填）

| 参数 | ENV | 说明 |
|------|-----|------|
| `app_secret` | `APP_SECRET` | **必填**。用于会话加密、OAuth client_secret 加密、数据源密码加密 |

未设置 `APP_SECRET` 时服务无法启动。

## 服务器配置 (ServerConfig)

| 参数 | 类型 | 默认值 | ENV | 说明 |
|------|------|--------|-----|------|
| `host` | string | `localhost` | `SERVER__HOST` | 监听地址 |
| `port` | integer | `18081` | `SERVER__PORT` | 监听端口 |
| `transport` | string | `stdio` | `SERVER__TRANSPORT` | `stdio` 或 `http` |
| `static_path` | string | `{cwd}/resources/web` | `SERVER__STATIC_PATH` | Web 控制台静态资源 |
| `docs_path` | string | `{cwd}/resources/docs` | — | 内置文档路径（仅代码默认值） |
| `cli_path` | string | `{cwd}/resources/cli/latest` | — | CLI 二进制分发目录 |
| `base_url` | string | `/dm-mcp` | `SERVER__BASE_URL` | API 与 Web 基础路径 |
| `frontend_url` | string | `""` | `SERVER__FRONTEND_URL` | 前端地址（可选） |
| `workers` | integer | `1` | `SERVER__WORKERS` | Worker 进程数 |
| `debug` | boolean | `true` | `SERVER__DEBUG` | 调试模式 |
| `audit_enabled` | boolean | `true` | `SERVER__AUDIT_ENABLED` | 是否启用审计 |

## 数据库配置 (DatabaseConfig)

| 参数 | 类型 | 默认值 | ENV | 说明 |
|------|------|--------|-----|------|
| `db_type` | string | `sqlite` | `DATABASE__DB_TYPE` | `sqlite` / `dameng` / `mysql` / `postgresql` |
| `echo` | boolean | `false` | `DATABASE__ECHO` | 是否打印 SQL |
| `pool_pre_ping` | boolean | `true` | `DATABASE__POOL_PRE_PING` | 连接池预检查 |
| `pool_recycle` | integer | `0` | `DATABASE__POOL_RECYCLE` | 连接回收时间（秒），0 表示不回收 |

### SQLite

| 参数 | 默认值 | ENV |
|------|--------|-----|
| `sqlite.db_path` | `server.db` | `DATABASE__SQLITE__DB_PATH` |

### 达梦

| 参数 | 默认值 | ENV |
|------|--------|-----|
| `dameng.host` | `localhost` | `DATABASE__DAMENG__HOST` |
| `dameng.port` | `5236` | `DATABASE__DAMENG__PORT` |
| `dameng.user` | `SYSDBA` | `DATABASE__DAMENG__USER` |
| `dameng.password` | `SYSDBA` | `DATABASE__DAMENG__PASSWORD` |
| `dameng.database` | `""` | `DATABASE__DAMENG__DATABASE` |

> `dameng.database` 留空时使用登录用户默认模式；设为 `DMMCP` 等名称时，该模式须已在达梦中创建。

### MySQL

| 参数 | 默认值 | ENV |
|------|--------|-----|
| `mysql.host` | `localhost` | `DATABASE__MYSQL__HOST` |
| `mysql.port` | `3306` | `DATABASE__MYSQL__PORT` |
| `mysql.user` | `root` | `DATABASE__MYSQL__USER` |
| `mysql.password` | `""` | `DATABASE__MYSQL__PASSWORD` |
| `mysql.database` | `DMMCP` | `DATABASE__MYSQL__DATABASE` |
| `mysql.charset` | `utf8mb4` | `DATABASE__MYSQL__CHARSET` |

### PostgreSQL

| 参数 | 默认值 | ENV |
|------|--------|-----|
| `postgresql.host` | `localhost` | `DATABASE__POSTGRESQL__HOST` |
| `postgresql.port` | `5432` | `DATABASE__POSTGRESQL__PORT` |
| `postgresql.user` | `postgres` | `DATABASE__POSTGRESQL__USER` |
| `postgresql.password` | `""` | `DATABASE__POSTGRESQL__PASSWORD` |
| `postgresql.database` | `DMMCP` | `DATABASE__POSTGRESQL__DATABASE` |

## 日志配置 (LoggingConfig)

| 参数 | 默认值 | ENV | 说明 |
|------|--------|-----|------|
| `level` | `INFO` | `LOGGING__LEVEL` | `DEBUG` / `INFO` / `WARNING` / `ERROR` / `CRITICAL` |
| `log_dir` | `logs` | `LOGGING__LOG_DIR` | 日志目录 |
| `enable_console` | `true` | `LOGGING__ENABLE_CONSOLE` | 控制台输出 |
| `enable_file` | `true` | `LOGGING__ENABLE_FILE` | 文件日志 |
| `enable_audit` | `true` | `LOGGING__ENABLE_AUDIT` | 审计日志 |
| `rotation` | `10 MB` | `LOGGING__ROTATION` | 日志切割 |
| `retention` | `30 days` | `LOGGING__RETENTION` | 保留时间 |
| `compression` | `zip` | `LOGGING__COMPRESSION` | 压缩格式，`null` 不压缩 |
| `audit_file` | `null` | `LOGGING__AUDIT_FILE` | 自定义审计日志路径（留空则自动生成） |

> 不要写 `LOGGING__AUDIT_FILE=`（空值会被错误解析）；不需要时请删除该行或注释掉。

## 指标配置 (MetricsConfig)

| 参数 | 默认值 | ENV | 说明 |
|------|--------|-----|------|
| `enabled` | `false` | `METRICS__ENABLED` | 是否启用 |
| `multiproc_dir` | `metrics` | `METRICS__MULTIPROC_DIR` | Prometheus 多进程目录 |
| `http_port` | `3001` | `METRICS__HTTP_PORT` | 指标端口 |
| `http_path` | `/metrics` | `METRICS__HTTP_PATH` | 指标路径 |

## 运行时 API 配置

以下配置**不再**通过环境变量设置，而是通过 REST API 管理并持久化到数据库：

### 认证配置

- `GET/PUT /api/v1/auth-config` — OAuth 全局开关、JWT 过期时间等
- `GET/PUT /api/v1/oauth/providers/{slot}` — OAuth Provider 配置

### 连接池行为 (DmPoolConfig)

- `GET/PUT /api/v1/pool-config`

| 字段 | 默认值 | 说明 |
|------|--------|------|
| `read_write_split` | `true` | 读写分离 |
| `load_balancing_strategy` | `least_connections` | `round_robin` / `least_connections` / `weighted_round_robin` |
| `default_source` | `primary` | 默认数据源 |
| `max_retries` | `1` | 最大重试次数 |
| `retry_backoff_ms` | `100` | 重试退避（毫秒） |

### 数据源

通过 `/api/v1/datasources` 相关接口创建、更新、启用/禁用，或在 [控制台](./console/datasource) 页面操作。

SQL 黑名单关键字在代码内静态配置，不可通过环境变量修改。

## 配置示例

### 命令行

```bash
uv run dm-mcp-server --server.port 8080 --server.transport http --logging.level DEBUG
```

### 环境变量

```bash
export APP_SECRET=your-long-random-secret
export SERVER__PORT=8080
export SERVER__TRANSPORT=http
export LOGGING__LEVEL=DEBUG
uv run dm-mcp-server
```

### .env 文件

```env
APP_SECRET=your-long-random-secret
SERVER__PORT=8080
SERVER__TRANSPORT=http
SERVER__HOST=0.0.0.0
LOGGING__LEVEL=DEBUG
DATABASE__DB_TYPE=dameng
DATABASE__DAMENG__HOST=192.168.1.100
DATABASE__DAMENG__PORT=5236
DATABASE__DAMENG__USER=SYSDBA
DATABASE__DAMENG__PASSWORD=your-password
```

### 混合覆盖

`.env` 设置默认值，命令行覆盖特定项：

```bash
# .env: SERVER__PORT=18081, LOGGING__LEVEL=INFO
uv run dm-mcp-server --server.port 8080 --logging.level DEBUG
# 结果: port=8080, level=DEBUG, 其余取自 .env
