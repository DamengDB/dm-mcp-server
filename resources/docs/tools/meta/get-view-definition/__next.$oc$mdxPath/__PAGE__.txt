5:T557,# get_view_definition <ToolBadge />

返回视图的完整 CREATE VIEW DDL 文本。
需要视图的完整定义或依赖关系；schema 和 view 必填。

## 参数

| 名称 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| schema | str | None | None | 视图所属 schema 名称（必填）。 |
| view | str | '' | 视图名（必填）。 |
| view_comment | str | None | None | 若已获取，可透传避免重复查询。 |

## SQL: Get View Info

```sql
SELECT
  S.NAME AS SCHEMA_NAME,
  T.NAME AS VIEW_NAME,
  'VIEW' AS OBJECT_TYPE,
  TC.COMMENT$ AS VIEW_COMMENT
FROM SYS.SYSOBJECTS T
JOIN SYS.SYSOBJECTS S ON T.SCHID = S.ID
LEFT JOIN SYS.SYSTABLECOMMENTS TC
  ON S.NAME = TC.SCHNAME
 AND T.NAME = TC.TVNAME
 AND TC.TABLE_TYPE = 'VIEW'
WHERE S.NAME = ? AND T.NAME = ?
  AND T.TYPE$ = 'SCHOBJ' AND T.SUBTYPE$ = 'VIEW'
```

## SQL: Get View Definition

```sql
SELECT TEXT AS DEFINITION
FROM DBA_VIEWS
WHERE OWNER = ? AND VIEW_NAME = ?
```

## 输出示例

```json
{
  "schema": "SALES",
  "view": "V_ORDER_SUMMARY",
  "comment": "订单汇总视图",
  "definition": "CREATE VIEW \"SALES\".\"V_ORDER_SUMMARY\" AS\nSELECT CUSTOMER_ID, COUNT(*) AS ORDER_COUNT, SUM(TOTAL_AMOUNT) AS TOTAL_AMOUNT\nFROM SALES.T_ORDER\nGROUP BY CUSTOMER_ID"
}
```

dmctl meta get_view_definition --schema None --view  --view_comment None
