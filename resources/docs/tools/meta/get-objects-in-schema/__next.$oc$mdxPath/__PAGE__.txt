5:T564,# get_objects_in_schema <ResourceBadge />

通过 MCP Resource URI 获取指定 schema 下的表和视图列表（含 comment 摘要）。

适合在 AI 询问"某个 schema 里有哪些表"时自动触发。

## Resource URI

```
dm://schema/{schema}
```

| 占位符 | 说明 |
|--------|------|
| schema | 目标 schema 名称（如 SYSDBA）。 |

## 输出示例

```json
{
  "schema": "SALES",
  "schema_info": {
    "SCHEMA_NAME": "SALES",
    "OWNER": "SYSDBA"
  },
  "tables": {
    "columns": ["schema_name", "table_name", "table_type", "comment"],
    "records": [
      ["SALES", "T_ORDER", "TABLE", "订单主表"],
      ["SALES", "T_CUSTOMER", "TABLE", "客户信息表"],
      ["SALES", "T_PRODUCT", "TABLE", ""]
    ]
  },
  "views": {
    "columns": ["schema_name", "table_name", "table_type", "comment"],
    "records": [
      ["SALES", "V_ORDER_SUMMARY", "VIEW", "订单汇总视图"],
      ["SALES", "V_SALES_MONTHLY", "VIEW", ""]
    ]
  }
}
```
