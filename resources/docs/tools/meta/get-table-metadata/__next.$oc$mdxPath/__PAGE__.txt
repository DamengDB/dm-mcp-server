5:T60b,# get_table_metadata <ResourceBadge />

通过 MCP Resource URI 获取表的结构概览：列信息、索引、所有者及创建时间。

适合在 AI 对话中自然引用表结构时自动触发，无需显式调用工具。

## Resource URI

```
dm://table/{schema}/{table}
```

| 占位符 | 说明 |
|--------|------|
| schema | 表所属 schema 名称（如 SYSDBA）。 |
| table | 表名。 |

## 输出示例

```json
{
  "schema": "SALES",
  "table": "T_ORDER",
  "table_info": {
    "SCHEMA_NAME": "SALES",
    "TABLE_NAME": "T_ORDER"
  },
  "columns_preview": {
    "columns": ["column_name", "data_type", "nullable", "comment"],
    "records": [
      ["ORDER_ID", "INT", false, "订单ID"],
      ["CUSTOMER_ID", "INT", false, "客户ID"],
      ["ORDER_DATE", "DATE", false, "下单日期"],
      ["STATUS", "VARCHAR", true, "订单状态"],
      ["TOTAL_AMOUNT", "DECIMAL", true, "订单总金额"]
    ]
  },
  "indexes_preview": [
    {"name": "PK_ORDER", "type": "UNIQUE", "columns": [{"name": "ORDER_ID", "sort_order": "ASC"}]},
    {"name": "IDX_ORDER_DATE", "type": "NORMAL", "columns": [{"name": "ORDER_DATE", "sort_order": "ASC"}]}
  ]
}
```
