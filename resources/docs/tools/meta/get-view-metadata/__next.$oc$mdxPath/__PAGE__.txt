5:T5a3,# get_view_metadata <ResourceBadge />

通过 MCP Resource URI 获取视图的结构概览：列信息、定义摘要及 comment。

适合在 AI 对话中自然引用视图结构时自动触发。

## Resource URI

```
dm://view/{schema}/{view}
```

| 占位符 | 说明 |
|--------|------|
| schema | 视图所属 schema 名称。 |
| view | 视图名。 |

## 输出示例

```json
{
  "schema": "SALES",
  "view": "V_ORDER_SUMMARY",
  "view_info": {
    "SCHEMA_NAME": "SALES",
    "VIEW_NAME": "V_ORDER_SUMMARY",
    "OBJECT_TYPE": "VIEW",
    "COMMENT": "订单汇总视图"
  },
  "definition_preview": "CREATE VIEW \"SALES\".\"V_ORDER_SUMMARY\" AS\nSELECT CUSTOMER_ID, COUNT(*) AS ORDER_COUNT...",
  "columns_preview": {
    "columns": ["column_name", "data_type", "nullable", "comment"],
    "records": [
      ["CUSTOMER_ID", "INT", false, null],
      ["ORDER_COUNT", "INT", true, null],
      ["TOTAL_AMOUNT", "DECIMAL", true, null]
    ]
  }
}
```
